About TasteScape Project

    Food Waste Overview

    Metrics have been provided by voedingscentrum.nl (Netherlands Nutrition Centre)

    Problem Context

    Consumers play a significant role in generating food waste, accounting for a substantial share in total food waste, ranging between 23% and 32%. Addressing this issue calls for solutions that resonate with consumers, enabling them to alter their behaviors.

    In the Netherlands, approximately 80% of individuals express willingness to reduce their food waste. Coupled with the fact that most households frequent supermarkets 2-3 times a week, there's a clear opportunity to develop a tool that assists in meal planning while being sustainable.

    In the graph, we can see a total amount of 34.3 kg per person per year is wasted. The combination of fruits and vegetables takes up a significant portion of 6.7 kg per person per year, or nearly 20% of the total food waste.

    Methods to reduce food waste

    The Dutch Nutrition Centre emphasizes these methods to reduce food waste for consumers and households.

    • Make a shopping list. This can account for a great reduction in food waste.
    • Prepare the right amount of food by measuring the ingredients.
    • Understand the importance of looking, smelling and tasting before throwing out products.
    • Be informed about correct storage methods.
    • Increase food skills by preparing meals using leftover products.

    Sustainable Development Goals (SDGs)

    The reduction of food waste works to directly affect the SDG Goal 12. Goal 12 aims to "ensure sustainable consumption and production patterns."

    With food waste being a large contributor to the global waste problem, SDG 12.3 states: “halve per capita global food waste at the retail and consumer level, and reduce food losses along production and supply chains by 2030”.

    Usage Instructions

    Please refer to this for proper usage of the tool.

    Search Engine and Image Detection

    The search function can be used to search for any type of ingredient. When searching for multiple ingredients, make sure to seperate them with a comma followed by a space. For example: 'thyme, onion, egg'. The search function can accept an infinite amount of ingredients.

    For the image detection tool, instructions are available at the 'demo' page. The detection model accepts PNG and JPG file formats. The tool performs best when ingredients are seperated from eachother, without overlapping.

    Model Summary

    A custom YOLOv8 model has been trained and implemented. Key performance metrics are available below.

    YOLOv8 - Ultralytics' Solution for Object Detection

      The YOLO (You Only Look Once) series of models has become famous in the computer vision world. YOLO's fame is attributable to its considerable accuracy while maintaining a small model size. YOLOv8 is the newest state-of-the-art YOLO model that can be used for object detection, image classification, and instance segmentation tasks.

      Development Challenges

      Several challenges have been encountered while developing the tool. The main challenge being the performance of the model. By using images where the setting is almost always optimal, for example perfect lighting, makes the model perform exceptionally well on paper, while possibly lacking in real-life use cases.

      Another issue has been over- and under-representation of classes and annotations. Smaller fruits and vegetables are over-represented by their frequent appearances and a high amount of annotations per image.

      Tools and Experience

      The following tools have been used to design and deploy the TasteScape Project.

      Python Extensions

      This is a Flask-based web application where all functionalities are powered by Python, including the model training, while the pages and design elements are implemented separately.

      SQLite Database

      To store the information gathered from data mining and processing through Web Scraping, a SQLite database has been created, containing all of the recipe data.

      Computer Vision

      The tool uses a custom YOLOv8 model, trained on over 40 classes of different fruits and vegetables. The image dataset contains over 8,000 images.

      Problem Statement

      By searching the market and challenges we face as a society, the initial idea of this project was created. This project aims to help with a data-driven approach to the problem.

      Web Scraping

      Recipe and ingredient data have been gathered from New York Times Cooking with Beautifulsoup4. The nature of the website allowed for uncomplicated, structured scraping.

      Data Processing

      The gathered data has been processed with regular expression patterns and other methods to ensure quality data is present within the database.

      HTML and CSS

      Originally, this website was based on a template. Along the way, it has been extensively modified to provide a smooth experience. The ease of access aims to bring humans and AI closer together.

      Humans and AI

      The primary goal of this web application is to bridge the gap between end-users and AI, while contributing to society by working on the challenge of food waste.

      Feature Considerations

      These concepts are being considered for further development of the project.

      • Creation of pages per noteworthy ingredient with limited shelf life.
      • Informing users about nutrition facts for these ingredeints.
      • Informing users about the correct storage methods of ingredients.
      • Creating a shopping list through the application.
      • Automated online grocery shopping based on selected recipes.
      • Improvement of search mechanic by using tags such as vegan, vegetarian, brunch, lunch.
      • Tools to adjust recipes based on dietary restrictions.
      • Tools to suggest substitutions when nearly all ingredients are present.
      • Implementation of an OCR model to detect canned and packaged ingredients.
      • Addition of user profiles, without the 'blog' aspect.
      • Ability to upload recipes. Possibly allowing comments on recipes.
      • Feature to mark recipes as 'favorite' or 'completed'.
      • Feature to add ratings to recipes.
